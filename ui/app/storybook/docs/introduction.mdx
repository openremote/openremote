# OpenRemote UI documentation
<br />

This website will showcase components and patterns used in OpenRemote UI.<br />
It provides information about the design system, component API and give code examples.<br />
Using these components makes interacting with OpenRemote easier and consistent.

### Features
- Works with all frameworks
- Open Source, and available for free on [NPM](https://www.npmjs.com/org/openremote)
- Extensive set of components (20+)
- Easy integration with OpenRemote and its APIs
